What is Google?

We probably all know Google unless you’ve been living under a rock since 1996. Google’s flagship product is its search engine. This is what we will be reviewing. Recently it was found guilty of paying Apple a very large amount of money for it to be the iPhone's default search engine. Hence they are now declared as a monopoly.

Strengths

Huge Index of Web Pages

Being a search engine requires that you have web pages that people want to see. This is probably simpleish. As we could start at any Wikipedia webpage, and save the contents of the page. Then go to every site that that Wikipedia page references and repeat. This works until it comes to displaying the search results. That would get difficult. But, because Google has a huge index of web pages, they can show you a lot of relevant search results.

Advanced Algorithm

This is the hard part that I said in the ladder point. To build one like Google’s algorithm would require insane power to train the AI. When searching the web, Google will run this algo on your search and Google account, I assume, and return the results from the search.

Image Search

Using image search is so fun. For example, you could take a picture of your fridge and it would show relevant and similar models of your fridge. This would be a very hard AI to build without the vast amounts of data that Google has.
